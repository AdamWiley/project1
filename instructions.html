Twenty-One Pickup

Description

Twenty-One Pickup is a two-player game that starts with a pile of 21 stones.

Each player takes turns removing 1, 2, or 3 stones from the pile.

The player that removes the last stone wins the game.

Several requirements are given for our version of the game.

Top-Level Pseudocode

  • Print Instructions
  • Ask the user who should go first, with the option to quit
  • While the user does not quit:
    • Create a Game object and initialize it with the starting player
    • Play the game
    • Show the winner
    • Ask the user who should go first for the next game, with the option to quit

Requirements

You will need to write a class called Game to control the flow of the game.
This class should have members that:

  • Keep track of the current player
  • Keep track of the number of stones on the board

At the start and after each move the number of stones should be displayed
This class should have at least these member functions:

  • A constructor that initializes the game with a starting player
  • A method to start the game
  • A function for the user's move
    • Get the move from the keyboard
    • Remove the stones from the pile
    • Print the user's move to the console
  • A function for the user's input
    • Ask the user for the number of stones
    • Validate the input is correct
    • Return the number of stones for the move
  • A function for the computer's move
    • If three or fewer stone remain the computer picks them all up
    • If more than three stones remain, try to leave the pile with a multiple of four stones (think mod)
    • If that is not possible, remove one stone
    • Print the computer's move to the screen
  • A method to check if the game has a winner
  • A method to display the winner to the screen
  • A method to end the game

Grading

Your program will be tested for the following conditions:

  • The user starts the game
  • The computer starts the game
  • The user inputs 1, 2, 3, and an illegal move
  • The computer plays 1, 2, or 3 moves 
  • (this will be based off of the above logic)
  • The user wins the game
  • The computer wins the game
